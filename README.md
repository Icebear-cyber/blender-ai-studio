# Blender AI Studio

AI-powered Blender studio for generating and automating 3D content using artificial intelligence.

## Overview

Blender AI Studio integrates cutting-edge AI models with Blender to help artists, designers, and developers automate and enhance their 3D workflows. Generate textures, models, animations, and scenes using natural language prompts and AI-driven tools.

## Features

- **AI-Powered Generation**: Create 3D assets using AI prompts
- **Blender Integration**: Seamlessly works within Blender's environment
- **Automation Scripts**: Automate repetitive 3D tasks with Python
- **Texture Generation**: AI-generated PBR textures for materials
- **Scene Setup**: Intelligent scene composition and lighting

## Tech Stack

- Python
- Blender Python API (bpy)
- AI/ML integrations

## Getting Started

```bash
git clone https://github.com/Icebear-cyber/blender-ai-studio.git
cd blender-ai-studio
```

Open Blender and load the scripts from the repository.
